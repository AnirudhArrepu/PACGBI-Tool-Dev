# PACGBI-Tool-Dev

## Getting Started
Follow these steps to set up the project locally:

1. Clone the Repository:
    ```
    git clone <repo name>
    cd PACGBI-Tool-Dev
    ```

2. Install required dependencies:
    ```
    pip install -r requirements.txt
    ```
    
3. Run the pipeline.py file to execute the code via the terminal:
    ```
    python3 pipeline.py
    ```
